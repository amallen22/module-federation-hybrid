login
============

Sign In, Sign Up, Sign Out and Forgotten password featured SPA application.


## Install ##

`npm install`


## Development tips ##

Command to update distribution code one time only:
`$ npm run build`

Command to permanently update distribution code:
`$ npm run dev`
